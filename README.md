# HTML Audio & Video Player

![Project Preview](preview.jpg)

This repository contains the solution for the **"HTML Audio and Video Player"** lab. The project demonstrates how to embed multimedia content into a webpage using native HTML5 elements.

## Project Objective
Build a web page that includes two sections: one for a video player and one for an audio player, fulfilling specific user stories for layout and functionality.

## Features
* **Video Player:**
    * Displays a coding tutorial video (`.mp4`) .
    * Fixed width of **640px**.
    * Includes native **controls** (play/pause/volume).
* **Audio Player:**
    * Plays a music track ("Sailing Away").
    * Includes **controls** and set to **loop** automatically.
* **Structure:**
    * Uses semantic `<section>` tags to separate the media players.

## Technologies Used
* HTML5
